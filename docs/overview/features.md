# 功能特性

Flovart 更推荐按创作链路使用，而不是把每个能力拆开记：

## 🎨 创作工作流

| 工作流 | 你会怎么用 |
| ------ | ---------- |
| **导入参考图** | 把角色、场景、产品或草图拖进画布，作为后续生成的视觉锚点。 |
| **@ 引用画布节点** | 在节点下方的 PromptBar 输入 `@`，直接把画布里的图片、视频或文本节点作为上下文。 |
| **生成 4 个方案** | 围绕同一组参考图批量生成 2/4 个方向，快速比较构图、风格和细节。 |
| **A/B 对比与局部修正** | 用对比视图挑选结果，再对选中图片做局部重绘、扩图、去背景、超分或滤镜调整。 |
| **保存为素材** | 把满意的角色、场景、道具沉淀进素材库，后续直接拖回画布复用。 |
| **让 Agent 继续扩展** | 外部 Agent 可以通过 CLI / SKILL 读取画布、点火节点、重试失败任务，并继续扩展镜头或视频。 |

## ⚡ 核心功能

| 功能 | 说明 |
| ---- | ---- |
| **Canvas / Workflow 切换** | 底栏一键切换画布模式 ↔ 节点流模式，共享同一画布状态 |
| **无限画布** | 缩放拖拽、画笔、形状、文本、箭头、图层管理、智能对齐 |
| **AI 文生图** | 提示词生成图片；支持 Gemini / DALL-E / SDXL 等 |
| **AI 图片编辑** | 选中图片 + 提示词 → 局部重绘、去背景、超分辨率 |
| **AI 扩图** | 选择方向自动延展图片内容 |
| **AI 文生视频** | Veo / Sora 文生视频，支持多种宽高比 |
| **AI Agent** | 多角色 Agent 群聊（创意总监、提示词工程师、风格大师等），讨论后自动生图 |
| **滤镜 / 调色** | 实时亮度、对比度、饱和度、色调、模糊、复古等 |
| **图层蒙版** | 画笔擦除 / 恢复的无损蒙版 |
| **批量生成** | 一次生成 2/4 个变体并对比选择 |
| **提示词润色** | 一键 LLM 自动优化提示词 |
| **@ 引用** | 输入框输入 `@` 引用画布元素作为参考图 |
| **角色锁定** | 锁定角色外观，后续生成保持一致 |
| **素材库** | 角色/场景/道具分类管理，拖入画布复用 |
| **多 Provider** | Google、OpenAI、DeepSeek、MiniMax、火山引擎、Qwen 等 12+ Provider |
| **第三方适配** | 持续适配 OpenAI-compatible 中转站/聚合端点，兼容规则持续扩充 |
| **凭据自动检测** | 粘贴服务凭据自动检测 Provider + 拉取可用模型 |
| **A/B 对比** | 拖动滑块对比两张图片 |
| **双语** | 简体中文 / English UI 自由切换 |
| **亮暗主题** | 亮色 / 暗色主题适配，Workflow UI 完全支持 |

## 🔌 Agent 集成

- **CLI 控制面**：`tools/flovart/cli.js` 提供 30+ 确定性命令
- **SKILL.md**：Claude Code / OpenCode / Codex 可直接驱动
- **命令队列**：浏览器轮询 `.flovart/command-queue.json` 执行生成任务
- **密钥隔离**：CLI 不读取、不输出、不保存 API Key

## 🎯 技术栈

- **前端框架**：React 19 + TypeScript 5.8 + Vite 6
- **状态管理**：Zustand + TanStack Query
- **UI 库**：shadcn/ui + Radix UI
- **样式系统**：Tailwind CSS + Framer Motion
- **桌面端**：Tauri 2
- **浏览器扩展**：Plasmo Framework

## 📦 部署方式

- ✅ **在线 Demo** — GitHub Pages 托管
- ✅ **本地运行** — npm run dev
- ✅ **Docker** — docker-compose 一键部署
- ✅ **Tauri 桌面端** — Windows / macOS / Linux
- 🔜 **浏览器扩展** — Chrome / Edge 商店（准备中）
