## 💡 技术要点

### Konva.js视频处理核心代码
```tsx
// 1. 创建video元素
const video = document.createElement('video');
video.src = videoUrl;

// 2. 作为Image节点渲染
<KonvaImage
  image={video}  // 直接传入video元素
  x={x} y={y}
  width={width} height={height}
  draggable
/>

// 3. 动画循环更新
const anim = new Konva.Animation(() => {
  layer.batchDraw();  // Konva自动从video提取当前帧
}, layer);
anim.start();
```
